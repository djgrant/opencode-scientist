## Orientation

Check your project for:
- Test runner configuration (jest, vitest, bun test, etc.)
- Existing test patterns and utilities
- Integration vs unit test locations

## Writing Tests

- Follow existing patterns in your project's test files
- Use the project's established test runner
- Look for shared test utilities before creating new ones

## Browser Testing

For browser-based testing, load the `browser` skill.
